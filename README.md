# Japan 2026 Trip Planner

Interactive route and planning files for a public-transport-only trip through Japan from June 12 to July 12, 2026.

## Open The Planner

Download this repository, then open `japan-route-map.html` in a browser.

The map includes:

- Main stops, dates, and nights
- Approximate train, bus, ferry, and flight paths
- Estimated travel times and costs
- Editable local notes, attraction ideas, and accommodation links
- A linked packing checklist

Planner notes are saved in the browser's local storage and are not automatically committed to this repository.

## Files

- `japan-route-map.html` - interactive route map and trip planner
- `packing-checklist.html` - interactive packing checklist
- `ITINERARY.md` - portable snapshot of the master itinerary
